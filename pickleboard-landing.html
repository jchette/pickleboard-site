HOW IT WORKS

01
Install on your server

Upload PickleBoard to your existing web hosting. No special server required — runs on any shared PHP host including Bluehost and GoDaddy.

02
Connect CourtReserve

Enter your CourtReserve Org ID and API key in the admin panel. Map each court to its CourtReserve ID. Done — live data starts flowing immediately.

03
Point your TVs

Navigate each court TV browser to its unique URL. Press F11 for fullscreen kiosk mode. The display auto-refreshes — no further interaction needed.

04
Manage everything

Upload ads, write announcements, adjust settings — all from the web-based admin panel. Changes go live on every TV within 60 seconds.

Yes — PickleBoard is built specifically for facilities running CourtReserve. It connects directly to the CourtReserve API to pull live booking and event data. You'll need your CourtReserve Org ID and API key to enable live data. Demo mode is available without an API connection.
Any shared web hosting with PHP support works — Bluehost, GoDaddy, SiteGround, and similar hosts are all compatible. If you already have a website, you can install PickleBoard in a subfolder on the same hosting account at no additional cost.
Any smart TV with a built-in browser works — Samsung, LG, Fire TV, Chromecast with Google TV, or a cheap Amazon Fire Stick. Open the browser, navigate to the court URL, press F11 for fullscreen, and you're done. No apps to install, no accounts to create.
PickleBoard is a one-time license fee per facility. There are no monthly charges, no renewal fees, and no per-court charges beyond the initial tier. Once you purchase, it's yours to run as long as your hosting is active.
