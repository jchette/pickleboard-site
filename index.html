How it works

Deploy in an afternoon.
Earn from day one.

No database, no developer needed. PickleBoard runs on any PHP web host and talks directly to CourtReserve's API.

1
📁
Upload files to your host
Drop the PickleBoard files into any PHP-enabled web host — shared hosting, VPS, or your existing WordPress server. No database needed.
2
🔧
Configure via the Admin Panel
Log in to the admin panel, enter your CourtReserve Org ID and API key, name your courts, upload your club logo, and set up the booking QR code.
3
📺
Open the URL on each TV
Navigate to your court URL on a browser-enabled TV or Fire Stick and press F11 for fullscreen. Courts update automatically — no further setup required.
4
🖼
Upload sponsor ads
From the Advertisements tab, upload images, videos, or paste a YouTube/Vimeo URL. Set display schedules, target specific courts, and track impressions.
5
💰
Sell ad slots to local businesses
Reach out to nearby restaurants, gyms, sporting goods stores, and physical therapists. Charge monthly for guaranteed impressions on your court TVs.
6
📊
Track & report performance
The Reports tab shows impression counts broken down by occupied vs. idle courts per advertiser — the data you need to justify renewals and rate increases.

Quick Setup

Running in under an hour.

No developer required. If you can manage a WordPress site, you can deploy PickleBoard.

1
Purchase your license & download PickleBoard
After purchase you'll receive a license key and a download link. Your key is tied to your installation domain, verified automatically on startup.
2
Upload to your web host
FTP or cPanel File Manager into your web host. Create a folder like yoursite.com/pickleboard/ and upload all files. For multiple facilities, create one folder per location.
3
Open the Admin Panel and run through Setup
Navigate to yoursite.com/pickleboard/#admin and log in (default password in config.json). The Setup tab walks you through 6 numbered steps: Demo Mode → Facility Name → Logo → Courts → CourtReserve API → QR Code.
4
Enter your license key
Paste your license key in Settings → License. PickleBoard will verify it against our server and unlock all features. No internet-restricted TV? The admin panel runs on any device — it just needs to phone home once.
5
Open court URLs on your TVs
Navigate to yoursite.com/pickleboard/#court/1 on Court 1's TV, press F11 for fullscreen. Repeat per court. TVs auto-refresh and pick up settings changes within 30 seconds.
6
Upload your first ads & start selling
Head to the Advertisements tab, upload your first sponsor's image or paste their YouTube URL, set the schedule, and you're live. Use the Reports tab to show sponsors their impression counts.
FAQ

Common questions.

What web hosting do I need?
Any PHP 7.4+ host works — shared hosting, a VPS, or your existing WordPress server. No database or special extensions are needed. The signage folder just needs to be writable by the web server.
Does it work with CourtReserve's check-in kiosks?
Yes. If your facility uses CourtReserve's check-in feature, PickleBoard can display check-in status badges next to player names. If you don't use kiosks, just toggle it off in the Setup tab — the API call is skipped entirely.
What TVs and devices does it work on?
Any device with a modern web browser — smart TVs, Amazon Fire TV Stick, Chromecast with Google TV, Roku via browser, or a mini PC. If it can open a URL full-screen, it works. No app install required.
How does the license key verification work?
When the admin panel loads, PickleBoard silently verifies your license key against our licensing server and ties it to your domain. TVs themselves don't need internet access to display — only the admin session verifies the key periodically.
Can I run multiple facilities from one license?
No — each physical facility location requires its own license. However, all locations can live on a single hosting account and server. Our Multi-Facility pricing gives you a discounted per-location rate when purchasing 3 or more licenses.
What happens if I change my domain?
License keys are tied to your installation domain. If you're migrating servers, contact support before moving files. Domain re-keying is free — just give us advance notice so we can transfer your key cleanly.
Do the TVs go offline if the internet drops?
PickleBoard includes an offline fallback — if the CourtReserve API is unreachable, the display shows the last known state and a soft error indicator. A 3 AM auto-reload keeps displays fresh without staff intervention.
Can I sell ad slots myself and keep 100% of the revenue?
Absolutely. PickleBoard doesn't take any cut of your advertising revenue. You own the relationship with your sponsors completely. The platform is a one-time tool cost — every dollar of ad revenue is yours.
